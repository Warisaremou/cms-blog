# CMS_BLOG API

### RUN THE PROJECT

Clone the project

```bash
  git clone https://github.com/Warisaremou/cms-blog.git
```

Go to the project directory

```bash
  cd cms-blog
```

Install dependencies

```bash
  npm install
```

Add .env file

```bash
  cp .env.example .env
```

Start the server

```bash
  npm run start
```

Run migration

```bash
  npm run migration
```

Run seeder

```bash
  npm run seeder
```

Generate a JWT secret key

```bash
  openssl rand -hex 48
```
